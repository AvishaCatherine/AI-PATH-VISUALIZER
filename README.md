## 🚀 Overview  

An interactive AI-based pathfinding visualizer that shows how algorithms explore and find the shortest path in real time.

---

## 🎯 Problem vs Solution  

| Problem | Solution |
|--------|---------|
| Hard to understand algorithms | Step-by-step visualization |
| Only final output visible | Real-time exploration |
| Hard to compare algorithms | Interactive comparison |

---

## ✨ Features  

| Feature | Description |
|--------|------------|
| 🎬 Animation | Real-time node exploration |
| 🧠 Algorithms | A*, BFS, DFS, Dijkstra |
| 🧩 Maze | Auto maze generation |
| ⚡ Speed | Adjustable speed |
| 🎯 Nodes | Custom start/end |

---

## 🧠 Algorithms  

| Algorithm | Shortest Path | Speed |
|----------|--------------|------|
