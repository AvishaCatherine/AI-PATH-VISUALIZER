## 🚀 Overview  

An interactive AI-based pathfinding visualizer that shows how algorithms explore and find the shortest path in real time.


## 🚀 Overview  

The **AI Path Visualizer** is an interactive system designed to demonstrate how Artificial Intelligence algorithms solve pathfinding problems in a structured environment.

It models a grid-based system as a **state space search problem**, where each cell represents a node and the objective is to find the most optimal path from a start point to a goal point.

Unlike traditional implementations that only display the final result, this project focuses on **visualizing the entire decision-making process** of algorithms in real time. Users can observe how nodes are explored, how paths are constructed, and how different algorithms behave under the same conditions.

The system integrates both **uninformed search algorithms** (BFS, DFS, Dijkstra) and **informed search algorithms** (A*), allowing users to compare efficiency, speed, and optimality.

Additionally, the project includes **maze generation techniques** and interactive controls, making it not just a visualization tool but a **learning platform for understanding core AI concepts** such as:
- State Space Search  
- Heuristic Functions  
- Problem-Solving Agents  
- Rational Decision Making  

This project bridges the gap between theoretical AI concepts and practical understanding by transforming algorithm behavior into an intuitive visual experience.
